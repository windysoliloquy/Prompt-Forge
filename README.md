# Prompt Forge

Prompt Forge is a native Windows desktop app for building high-quality image prompts through structured controls instead of manual prompt writing.

## Start Here

If you are new to Visual Studio or GitHub, use this order:

1. Open [PromptForge.sln](C:/Users/windy/OneDrive/Desktop/Prompt%20Forge/PromptForge.sln) in Visual Studio 2022.
2. Build and run the app with Ctrl+F5.
3. If you want to find the built .exe, read [WHERE_TO_FIND_THE_APP.md](C:/Users/windy/OneDrive/Desktop/Prompt%20Forge/WHERE_TO_FIND_THE_APP.md).

If the app has already been built, the executable is usually here:

PromptForge.App\bin\Debug\net8.0-windows\PromptForge.App.exe

## Quick Launch

1. Open [PromptForge.sln](C:/Users/windy/OneDrive/Desktop/Prompt%20Forge/PromptForge.sln).
2. Make sure PromptForge.App is the startup project.
3. Press F5 to run with debugging, or Ctrl+F5 to run without debugging.
